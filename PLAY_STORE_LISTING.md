# Google Play Store Listing

## App Details
- **Title**: AI APK Builder
- **Short Description**: Build Android apps from natural language prompts using AI
- **Full Description**: AI APK Builder is a powerful open-source tool that lets anyone create Android applications by simply describing what they want in plain English. Powered by advanced AI models (OpenAI, Groq, Ollama, and more), it generates complete, production-ready Android apps with Jetpack Compose UI, MVVM architecture, and Material3 design.

### Key Features
- Natural language app generation
- 18+ app type templates
- Multi-provider AI support (OpenAI, OpenRouter, Groq, Ollama)
- Real-time code generation with progress tracking
- Cloud builds via GitHub Actions, Codemagic, or local Docker
- APK/AAB download and sharing
- Project management and history
- QR code sharing for downloads
- Open source (MIT license)
