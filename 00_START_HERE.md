## 🎯 Quick Reference

### Key Files to Know

**Data Access**:
- `data/model/Models.kt` - All entities
- `data/local/AppDatabase.kt` - Database & DAOs
- `data/repository/ProjectRepository.kt` - Main repo

**Services**:
- `data/service/CodeGenerationService.kt` - AI generation interface
- `data/service/BuildProviderService.kt` - Build abstraction
- `data/service/BuildWorker.kt` - Background builds

**Utilities**:
- `util/ErrorHandling.kt` - Error types
- `util/ProjectGenerationUtils.kt` - Helpers
- `util/AppLogger.kt` - Logging
- `util/Extensions.kt` - 30+ helpers

**DI & Config**:
- `di/AppModule.kt` - Hilt configuration
- `AIAPKBuilderApp.kt` - App initialization

**UI Layer** (Phase 3):
- `ui/screens/` - Composable screens
- `viewmodel/` - MVVM ViewModels
- `ui/navigation/` - Navigation setup
