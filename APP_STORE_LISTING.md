## Title
AI APK Builder - Build Android Apps with AI

## Short Description
Generate production-ready Android apps from natural language prompts. 100% free, no subscriptions, no ads, no sign-in required.

## Full Description
AI APK Builder is a revolutionary open-source tool that transforms natural language descriptions into complete Android applications. Just describe what you want to build, and AI generates the full source code and compiled APK instantly.

### 🎯 Core Features
- **Natural Language to App**: Type "Create a weather app" and watch it build
- **18+ Templates**: Calculator, Chat, E-Commerce, Taxi, AI Assistant, Games, and more
- **Multiple AI Providers**: OpenAI, OpenRouter, Groq, Ollama (local), or custom APIs
- **Complete Export**: Download both APK (compiled) and source code (Kotlin + Jetpack Compose)
- **Production Ready**: MVVM architecture, Material Design 3, Dark/Light modes
- **Real-time Build Logs**: Watch the building process live

### 💪 Technical Excellence
- Built with Kotlin 2.0 and Jetpack Compose
- Material Design 3 with adaptive theming
- MVVM architecture with Hilt dependency injection
- SQLite local storage for project history
- 120+ unit tests with excellent code coverage

### 🏗️ Build Infrastructure
- **GitHub Actions**: Fast cloud builds (free tier)
- **Codemagic**: Alternative cloud provider
- **Docker**: Self-hosted builds on your server
- **Ollama**: 100% local processing (completely offline)

### 🔑 Key Features
- **Your Data, Your Privacy**: API keys stored encrypted on device
- **Offline Mode**: Works completely offline with Ollama
- **Project History**: Unlimited saved projects
- **QR Code Sharing**: Share generated apps via QR code
- **Dark Mode**: Full dark theme support
- **No Sign-In**: Starts working immediately
- **100% Free**: No subscriptions, no in-app purchases, no ads

### 💰 Pricing

## Troubleshooting

### Q: App says "No internet" but I'm online?
A: Check if your AI provider's API is reachable. Try switching providers.

### Q: Generated APK won't install?
A: Ensure you have Android 8.0+. Try installing the debug APK first.

### Q: Can I publish generated apps?
A: Yes! You own 100% of generated apps. Publish wherever you want.

### Q: Lost my API key?
A: Generate a new one from your AI provider's dashboard.
